# Crypto-portfolio-dashboard
A simple dashboard that track cryptocurrency portfolio value and price changes.

##Features 
-Track multiple cryptocurrency assets
-Monitor portfolio value in real time
-View price changes accross supported assets
-Basic analystics for crypto investments

##Technologies
-Python
-Crypto market APIs (CoinGecko or similar)
-Data visualization tools
-Web3 data integration

##Purpose
This project was created to explore crypto portfolio tracking and build a simple analytics dashboard for monitoring cryptocurrency investments.

##Future Improvemnts 
-Support for more cryptocurrencies
-Advanced portfolio analytics
-Real-time price alerts
-Intergation with crypto exchanges
